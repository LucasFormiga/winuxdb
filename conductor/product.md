# Initial Concept\n\nA website similar to ProtonDB, but for Windows apps on Linux, providing compatibility ratings (BORKED, BRONZE, SILVER, GOLD, PLATINUM, NATIVE) and multi-language support (English, Brazilian Portuguese, Spanish). The project aims for a unique, modern, and accessible design using customized shadcn components, following a mobile-first approach.
\n# Product Guide: WinuxDB\n\n## Vision\nWinuxDB is a modern, community-driven database focused on the compatibility of Windows applications on Linux systems. Similar to ProtonDB, it provides clear, reliable ratings for various applications running through Wine or having native Linux support.\n\n## Core Features\n- **Compatibility Ratings**: Categorization of apps as BORKED, BRONZE, SILVER, GOLD, PLATINUM (via Wine), or NATIVE.\n- **Multi-language Support**: Full support for English (default), Brazilian Portuguese, and Spanish with an easy-to-use language selector.\n- **Advanced Search & Discovery**:\n  - Search by application name.\n  - Filters: Rating (Medal), License, and Category.\n  - Sorting: Popularity, Release Date, and Rating.\n- **Rich App Listings**: Detailed displays including app name, logo/screenshot, and compatibility medal.\n- **Modern UI/UX**: A unique, high-quality, and accessible design built with customized shadcn components that avoids a "generic" look.\n- **Platform Strategy**: Mobile-first responsive design ensuring excellent usability across all devices.\n\n## Target Audience\n- Linux enthusiasts and new users looking for app compatibility information.\n- Developers and contributors interested in the Windows-on-Linux ecosystem.\n- Global community needing multi-lingual support for accessibility.
